
# Whisper (Windows) — Русский язык по умолчанию

Готовый пакет для Windows с Faster-Whisper. По умолчанию распознаёт **на русском** (можно поменять переменной `WHISPER_LANG`).

## 1) Что нужно
- **Python 3.9+**: https://www.python.org/downloads/  (во время установки поставьте галочку "Add Python to PATH")
- **FFmpeg** (для чтения аудио/видео):
  - Скачайте сборку (Essentials) отсюда: https://www.gyan.dev/ffmpeg/builds/
  - Распакуйте, добавьте путь к папке `bin` в переменную окружения **Path** (например, `C:\ffmpeg\bin`).
- (Опционально, для GPU) **драйвер NVIDIA** последней версии. CUDA Toolkit не обязателен.

## 2) Установка зависимостей
Откройте папку с проектом в Проводнике. Дважды кликните:
- `setup_venv.bat` — создаст виртуальное окружение и установит зависимости,
  затем пользуйтесь `start_cpu.bat` (CPU) или `start_gpu.bat` (GPU).

Либо вручную в терминале PowerShell/CMD:
```
python -m venv .venv
.\.venv\Scripts\activate
python -m pip install --upgrade pip
pip install -r requirements_windows.txt
```

## 3) Использование
1. Скопируйте ваши файлы в папку `audio` (поддержка .mp3, .wav, .m4a, .mp4, .flac, .ogg, .wma, .aac, .mkv).
2. Запустите:
   - `start_cpu.bat` — универсально, работает везде.
   - `start_gpu.bat` — быстрее на NVIDIA GPU (нужен свежий драйвер).
3. Готовый текст будет в папке `text` (с таймкодами).

## 4) Изменение модели или языка
- Модель: tiny | base | small | medium | large-v3 (см. комментарии в `.bat`).
- Язык разово: в окне перед запуском можно выполнить, например:
```
set WHISPER_LANG=en & start_gpu.bat
```
или просто отредактируйте `.bat` файлы.

## 5) Советы
- Если FFmpeg не найден — проверьте переменную Path (должна указывать на `...ffmpeg\bin`).
- Если при запуске `start_gpu.bat` видите ошибки CUDA/cuDNN — обновите драйвер NVIDIA.
- Для длинных записей GPU значительно быстрее, но CPU-режим (`int8`) тоже шустрый.

## 6) Веб-интерфейс
Для загрузки файлов через браузер запустите:
```
python web_app.py
```
После старта откройте [http://localhost:5000](http://localhost:5000) и загрузите аудиофайл. Распознанный текст появится на странице и сохранится в папке `text`.
